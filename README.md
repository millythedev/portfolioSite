### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/millythedev/portfolioSite.git
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Start the development server:
   ```bash
   bun run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
